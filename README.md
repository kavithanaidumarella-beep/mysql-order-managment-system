# Order Management System

## Project Overview

The Order Management System is a MySQL-based project used to manage customers, products, and orders. It helps in storing customer information, product details, and order records in an organized manner.

The system uses tables, stored procedures, triggers, and views to perform various operations such as adding customers, placing orders, updating stock automatically.

## Features

* Customer Management
* Product Management
* Order Management
* Stock Management
* Revenue Calculation

## Technology Used

* MySQL
